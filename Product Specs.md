# Product Spec

## Target User

Primary user: caregiver, parent, educator, or SLP setting up communication boards.

End user: nonspeaking or minimally speaking autistic child using visual communication support.

## First Use Case

A child taps visual buttons to communicate basic needs such as:

- help
- stop
- more
- yes
- no
- water
- bathroom
- hurt
- break
- play

## MVP Features

### Communication Mode

- Grid of picture/text buttons
- Text-to-speech output
- Sentence strip
- Clear button
- Starter boards

### Caregiver Mode

- Edit button labels
- Add custom buttons
- Choose board category
- Save locally

### Learning Mode: Future

- Match word to image
- Listen and choose answer
- Simple reading comprehension
- Sentence-building practice
- Writing practice
